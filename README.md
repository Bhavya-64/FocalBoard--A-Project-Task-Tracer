# FocalBoard – A Project Task Tracer

## Overview

FocalBoard – A Project Task Tracer is a project management and task tracking system inspired by tools like Kanban boards. It helps individuals and teams organize tasks, track progress, and manage project workflows efficiently.

The system allows users to create boards, manage tasks, track project progress, and collaborate with team members in a structured way.

---

## Features

* Task and project management
* Kanban-style workflow tracking
* Board creation for organizing tasks
* Status tracking (To Do, In Progress, Done)
* Team collaboration
* Structured task visualization

## Installation

### 1. Clone the repository

```
git clone https://github.com/Bhavya-64/FocalBoard--A-Project-Task-Tracer.git
```

### 2. Navigate to the project folder

```
cd FocalBoard--A-Project-Task-Tracer
```

### 3. Install dependencies

```
npm install
```

### 4. Run the project

```
npm start
```

---

## Usage

1. Create a new board.
2. Add tasks to the board.
3. Move tasks across workflow stages.
4. Track progress visually.
